# TripFlow 여행 관리

여행 전 준비, 여행 중 기록, 여행 후 정산과 리포트까지 관리하는 웹앱입니다.

## 주요 기능

- 여러 여행 관리
- 여행 전 준비물 체크리스트
- 항공, 숙박, 입장권 등 선결제 경비 기록
- 날짜별 일정 작성
- 오늘 화면과 여행지 날씨 확인
- 공용 경비, 개인 경비, 현지 지출 기록
- 현지 화폐와 원화 환산
- 여행 후 정산과 지출 리포트
- Supabase 클라우드 저장
- 공유용 HTML 내보내기

## GitHub Pages 배포
