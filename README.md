# Solo Trip Planner

혼자 여행을 계획하고, 여행 중 경비를 기록하고, 여행 후 리포트를 확인하는 웹앱입니다.

## 주요 기능

- 여러 여행 관리
- 날짜별 일정 작성
- 오늘 화면과 날씨 확인
- 공용 경비, 개인 경비, 선결제 경비 기록
- 현지 화폐와 원화 환산
- 여행 준비 체크리스트
- Supabase 클라우드 저장
- 공유용 HTML 내보내기

## GitHub Pages 배포

이 폴더의 파일을 GitHub 저장소에 올린 뒤, GitHub Pages를 `main` 브랜치의 root 폴더로 설정하면 됩니다.

Supabase 연결 방법은 `SUPABASE_SETUP.md` 파일을 참고하세요.
